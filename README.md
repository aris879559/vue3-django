# vue3-django
## vue3+django lyb project

## 功能
- 编辑、修改、删除和新增留言板信息功能
- 编辑、修改和新增信息提交后清空表单信息

###  vue3+Django实现前后端分离的留言板项目

### 后端django启动方式
```
python manage.py runserver 0.0.0.0:8080
```

### 前端 vue3启动方式
```
yarn dev
```
